Posicionando os Elementos em Tela

A propriedade position manipula a localização/posição do elemento na sua página.

  • position: static;
  • position: relative;
  • position: absolute;
  • position: fixed;
  • position: sticky;

Propriedade static

Faz com que os elementos sigam o fluxo normal da página, ou seja, a sequência em que estiverem os elementos no documento html é a sequência que serão exibidos na tela.

Propriedade relative

Visualmente se parece com a propriedade static, mas quando vamos definir um elemento com uma posição relativa, faz com que seja possível manipular mais 4 propriedades do elemento (top, bottom, left e right).

Exemplos:

Elemento 2:

      position: relative;
      bottom: 60px; /* afastou o elemento 2, 60px do fundo do bloco */
      left: 15px; /* afastou o elemento 2, 15px do elemento da esquerda */
    }

Elemento A
Elemento B

Elemento B:
    .relative-2 {
      position: relative;
      top: 60px; /* afastou o elemento B 60px do topo do bloco (ou seja, afastou 60px do elemento acima) */
      left: 130px; /* afastou o elemento B 130px da esquerda */
    }

Propriedade absolute

A propriedade absolute faz com que o elemento saia do fluxo normal, que foi declarado no html, ignorando totalmente os elementos/conteúdo ao seu redor, para que possamos posiciona-lo onde queremos de fato.

Exemplos:

Elemento 1
Elemento 2
Elemento 3

Elemento 2:
  .absolute {
    position: absolute;
    opacity: .5;
    left: 30px; /* afastou o elemento 2, 30px para a esquerda da página ignorando os outros elementos na linha*/
  }


Exemplo:
  html

  <div id="elemento-pai">
    <div id="elemento-filho" class="exemplo"></div>
  </div>

  css

  .exemplo {
    width: 100px;
    height: 100px;
    margin: 10px;
    padding: 10px;
    border-radius: 5px;
  }

  #elemento-pai {
    box-sizing: border-box;
    width: 50%;
    height: 300px;
    padding: 10px;
    border: 1px solid black;
    position: relative;
  }

  #elemento-filho {
    background-color: palegreen;
    position: absolute;
    top: 0;
    right: 0;
    margin: 0;
  }

Propriedade fixed

A propriedade fixed faz com que o elemento permaneça fixo na página.

Exemplo:

position: fixed;
  html

  <div id="elemento-1" class="exemplo">Elemento 1</div>

  css

  .exemplo {
    width: 100px;
    height: 100px;
    margin: 10px;
    padding: 10px;
    border-radius: 5px;
  }

  #elemento-1 {
    background-color: orange;
    position: fixed;
  }

Propriedade sticky

A propriedade sticky faz com que o elemento se comporte como se fosse um elemento de position: relative; em um determinado espaço, e em outro determinado espaço conforme a página dá scroll, como se fosse um elemento de position: fixed;.

Exemplo:

position: sticky;
    html

    <div id="elemento-7" class="exemplo">Elemento 7</div>

    css

    .exemplo {
      width: 100px;
      height: 100px;
      margin: 10px;
      padding: 10px;
      border-radius: 5px;
    }

    #elemento-7 {
      background-color: orange;
      position: sticky;
      bottom: 0;
    }
