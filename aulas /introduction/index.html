Cores

Existem várias formas de definirmos cores utilizando o CSS. São elas:

  • Cores pre-definidas
  • Palavra-chave "currentcolor"
  • Hexadecimal
  • Hexadecimal com transparência
  • RGB
  • RGBA
  • HSL
  • HSLA

HSL

hsl (hue, saturation, lightness)

  • Hue (matiz): é o grau na roda de cores (de 0 a 360 graus)
    • 0 ou 360 -> vermelho
    • 120 -> verde
    • 240 -> azul
  • Saturation (saturação): valor percentual
    • 0% -> tom de cinza
    • 100% cor total
  • Lightness (brilho): brilho / luminosidade da cor
    • 0% -> preto
    • 100% -> branco
