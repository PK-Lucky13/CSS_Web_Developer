Alterando Fundo dos Elementos

background-size: cover;

background-size: contain;

background-size: px ou %;

background-size: [altura] [largura];

background-repeat: repeat (auto);

background-repeat: repeat-x;

A imagem continua ocupando a altura e largura definidas para o elemento (div), porém ela se repetirá somente no eixo x (horizontal) do elemento.


background-repeat: repeat-y;

A imagem continua ocupando a altura e largura definidas para o elemento (div), porém ela se repetirá somente no eixo y (vertical) do elemento.


background-repeat: space;

Adciona espaço entre as imagens, repetindo elas nos eixos verticais e horizontais de modo que a visualização das mesmas não corte nenhuma imagem, ou seja, coloca um espaço entre as imagens sem cortar nenhuma delas.


background-repeat: round;

Repetido quantas vezes couber na área de posicionamento do plano de fundo. Se não couber um número inteiro de vezes, é redimensionado para que caiba.


background-repeat: no-repeat;

background-repeat: [valor eixo x] [valor eixo y];
