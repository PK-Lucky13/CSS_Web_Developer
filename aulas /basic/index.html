Seletores CSS

Usados para definir as propriedades CSS referenciando o elemento de acordo com sua necessidade:

  • Seletor por Tipo: Busca elementos por uma tag HTML específica (ex.: mark {cod CSS} <mark> conteúdo a ser estilizado </mark>, h1 {cod CSS} <h1> conteúdo a ser estilizado </h1>)
  • Seletor por id (#) - o id é único por elemento: Busca elementos através do atributo "id"
  • Seletor por class (.) - pode ser usada em mais de um elemento: Busca elementos através do atributo "class"
  • Seletor Universal (*) - mais usado para definir um padrão: Seleciona todos os elementos HTML
  • Seletor de Atributo ([atributo]): Seleciona elementos que possuem um atributo específico em nosso documento HTML e conseguimos também buscar atributos com um valor específico.

Agrupamento de Seletores

Podemos aplicar as mesmas regras CSS para vários seletores diferentes. Para isso, podemos separar os seletores utilizando uma vírgula (,).

Assim, não precisamos duplicar os mesmos códigos para todos, podemos criar um só bloco de CSS e reutilizar as regras desejadas.

Combinadores

Especifícia qual o tipo de relação entre 2 seletores, por exemplo entre 2 classes, ou uma classe e um id, ou uma tag e um seletor de atributo e assim por diante.

  1. Combinador descendente (espaço)
  2. Combinador filho (>)
  3. Combinador irmão adjacente (+)
  4. Combinador irmão em geral (~)
  5. Estrutura: [SELETOR] [COMBINADOR] [SELETOR]
Exemplo:

  • Item 1
    • Subitem A
    • Subitem B
  • Item 2
    • Subitem A
    • Subitem B
  • Item 3
    • Subitem A
    • Subitem B
  • Item 4
    • Subitem A
    • Subitem B
