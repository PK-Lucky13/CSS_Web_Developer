Objetivo geral

O objetivo geral deste curso é mostrar os conceitos de responsividade e ensiar as técnicas que podemos utilizar para deixarmos nosso site responsivo.

Pré-requisitos

É importante que você já tenha uma base de HTML e assistido ao Módulo 1 de CSS, juntamente com os cursos de Posicionamentos e Displays com CSS, Flexbox e Grid Layout
