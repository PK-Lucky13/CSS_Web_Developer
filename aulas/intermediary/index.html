Todo elemento possui um display?

Todo elemento HTML já vem com um valor de display como padrão

Cada elemento HTML já possui, por padrão, um valor da propriedade display já definidas. A tag <div>, por exemplo, possui um display em bloco, enquanto a tag <span> possui o valor inline como padrão.

Valores da propriedade display

  • display: block;
  • display: inline;
  • display: inline-block;*
  • display: none;*
display: block;

Preenche todo o espaço ocupado pela caixa na página, da esquerda para a direita, até o final.

Exemplos:

<div> Exemplo 1 </div>

<p> Lorem ipsum dolor sit amet consectetur adipisicing elit. Enim, earum dolore laboriosam error esse ipsum culpa magni saepe aliquam illo modi blanditiis libero. Corrupti, soluta eum quisquam quos tempora nobis. </p>

<span> Exemplo 3 </span> <span> Exemplo 3 </span>

<span style="display: block;"> Exemplo 3</span>
display: inline;

Os elementos serão alocados lado a lado, ocupando a mesma linha.

Exemplos:

<span> Exemplo inline 1 </span> <span> Exemplo inline 2 </span> <em> Exempo inline 3 </em> <em> Exempo inline 4 </em>

display: inline-block;

Visualmente é igual ao valor inline, com a diferença que com o inline-block é possível modificar a altura (heigth), largura (widht), margin e padding dos elementos.

display: flexbox; vs display: grid;

Display: flex

O flexbox é uma forma de organizar os elementos em uma página dentro de containers e de forma dinâmica, organizando os elementos em uma dimensão: ou horizontalmente (as linhas), ou verticalmente (em colunas).


Flexbox: 1 dimensão

➞

Display: grid

No CSS conseguimos trabalhar no nosso layout utilizando o conceito de grids. O grid vai organizar os elementos em 2 dimensões: de forma horizontal e vertical, definindo linhas e colunas para que os conteúdos da página sejam posicionados.


Grid Layout: 2 dimensões
