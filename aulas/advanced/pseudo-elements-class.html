Pseudo-elementos (::palavra-chave)

Usado para adicionar elementos extras na página sem a necessidade de declará-lo no HTML, ou seja, selecionar uma parte de algum elemento.

Identificado por ::palavra-chave

Pseudo-elemento - ::first-letter

Seleciona a primeira letra da primeira linha.

Propriedades disponíveis

  • color
  • background
  • margin
  • padding
  • border
  • font
  • Propriedades de Texto

Pseudo-elemento - ::first-line

Seleciona a primeira linha de um texto.

Propriedades disponíveis

  • color
  • background
  • font
  • Propriedades de Texto
